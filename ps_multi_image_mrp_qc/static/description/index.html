Take a Multi Pictures for Product QC

Enhance your Odoo 17 experience with our Multi Picture Product QC.

By default, Odoo provides support for attaching a single image per Quality Control (QC) checkpoint. To meet more comprehensive quality assurance needs, we have enhanced this functionality to allow for the attachment and management of multiple images per QC checkpoint.




Top Features

Discover the latest features and enhancements.

Take a Multi Picture

Users can now attach multiple images to a single QC checkpoint, allowing for a more detailed and thorough visual inspection process. Each image can represent different aspects of the product, enabling a comprehensive quality assessment.

Enhanced Image Management

The feature includes easy-to-use tools for uploading, downloading, and deleting multiple images directly from the QC Quality Checks and Shop Floor interface. The system maintains an organized gallery for each QC checkpoint, making it easier to review and analyze images.

Integration with Existing Workflows

The multi-image feature is fully integrated with existing QC workflows in Odoo, ensuring that it complements rather than complicates current processes.

Easy File Downloads

Quickly download any attached file directly from the preview interface with a single click. Enhance productivity by accessing necessary documents without navigating away from the form view.

Select Type as 'Take Multi Picture'

Select the type as "Multi Picture" to enable the ability to select multiple pictures during the quality check.

Select Multiple Image for QC

Clicking on "Take a Picture" allows users to select multiple images from their system. Once images are selected, the "Validate" button becomes visible. Users can delete selected images before validation if needed. Clicking "Validate" stores the selected images in the backend under Quality Checks.

Selected Image fetch on quality check.

Clicking "Validate" sends the selected images to the backend and stores them in the Quality Check record.
