# 制造订单剩余产品返回处理模块

## 功能描述

本模块为 Odoo 制造管理模块提供剩余产品返回处理功能。当制造订单选择"无欠单"时，系统会询问用户如何处理剩余产品。

## 主要功能

### 1. 剩余产品处理向导
- 当制造订单选择"无欠单"时，自动弹出处理向导
- 显示剩余产品数量和产品信息
- 提供两种处理方式选择

### 2. 返回位置选择
- **原材料不良品仓**：将剩余产品返回至不良品仓
- **主仓库**：将剩余产品返回至主仓库

### 3. 自动库存调拨
- 自动创建库存调拨单
- 支持记录返回原因
- 自动完成调拨流程

## 使用方法

1. **安装模块**：在 Odoo 应用商店中安装本模块
2. **配置仓库**：确保系统中配置了原材料不良品仓和主仓库
3. **使用功能**：
   - 在制造订单界面，当选择"无欠单"时
   - 系统会弹出剩余产品处理向导
   - 选择返回位置和填写原因
   - 确认后自动创建库存调拨

## 技术特点

- **用户友好**：直观的向导界面
- **自动化**：自动创建库存调拨单
- **可追溯**：记录处理原因和日志
- **灵活配置**：支持不同仓库位置配置

## 依赖模块

- mrp (制造管理)
- stock (库存管理)
