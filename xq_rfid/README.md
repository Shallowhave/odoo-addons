# RFID 标签管理模块

**版本**: 18.0.1.0.0  
**作者**: Grit  
**网站**: https://ifangtech.com

## 🎯 模块简介

为 Odoo 生产制造提供 RFID 标签管理和防伪溯源功能。

## ✨ 核心功能

- ✅ **RFID 标签管理** - 统一管理所有 RFID 标签
- ✅ **质检集成** - 通过质量检查点生成 RFID
- ✅ **序列号绑定** - 每个序列号对应唯一 RFID
- ✅ **生产溯源** - 记录生产订单、质检、日期
- ✅ **设备接口** - 预留 RFID 读写器接口

## 🚀 快速开始

### 1. 安装模块

```bash
./odoo-bin -i xq_rfid -d your_database
```

### 2. 创建质检点

- 进入：**质量 → 质量控制 → 质量检查点**
- 测试类型：选择 **RFID 标签**
- 配置产品和工序

### 3. 生成 RFID

- 生产过程中，质检对话框弹出
- 显示成品批次号和 RFID 状态
- 点击 **"通过"** 按钮
- 系统自动生成 RFID 标签
- 质检完成

## 🔑 关键特性

### ✅ 质检集成

```
生产订单 → 质检点 → 查看批次号 → 点击通过 → 自动生成 RFID → 完成质检
```

### ✅ 数据关联

```
RFID 标签
├── 序列号 (必填)
├── 产品
├── 生产订单
├── 质检记录
└── 生产日期
```

### ✅ 唯一性保证

- 一个 RFID 编号全局唯一
- 一个序列号只能有一个 RFID
- 自动检测重复，防止冲突

## ⚙️ 配置选项

### RFID 编号规则

默认: `RFID000001`, `RFID000002`, ...

可在 **设置 → 技术 → 序列** 中自定义。

### RFID 设备（可选）

如有硬件读写器：
1. 进入 **RFID → 配置 → RFID 设备配置**
2. 配置设备参数
3. 在质检点勾选"需要 RFID 设备"
