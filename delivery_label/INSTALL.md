# 发货标签模块安装说明

## 安装前准备

### 1. 安装Python依赖
在安装模块之前，请确保安装了以下Python包：

```bash
pip install Pillow>=8.0.0
pip install qrcode[pil]>=7.0.0
```

或者使用requirements.txt文件：
```bash
pip install -r requirements.txt
```

### 2. 检查Odoo版本
确保您的Odoo版本是18.0或更高版本。

### 3. 检查依赖模块
确保以下模块已安装：
- 库存管理 (stock)
- 销售 (sale) - 可选
- 发货 (delivery) - 可选

## 安装步骤

### 1. 复制模块文件
将整个 `delivery_label` 文件夹复制到您的Odoo addons目录中。

### 2. 更新应用列表
在Odoo中：
1. 进入"应用"菜单
2. 点击"更新应用列表"
3. 搜索"发货标签"

### 3. 安装模块
1. 点击"安装"按钮
2. 等待安装完成

### 4. 配置权限
安装完成后，需要为用户配置权限：
1. 进入"设置" > "用户和公司" > "用户"
2. 编辑用户，在"权限"标签页中添加"发货标签用户"或"发货标签管理员"组

## 故障排除

### 常见问题

1. **安装时出现依赖错误**
   - 确保已安装Pillow和qrcode库
   - 检查Python版本兼容性

2. **权限错误**
   - 确保用户有适当的权限组
   - 检查记录规则配置

3. **打印问题**
   - 检查PDF生成是否正常
   - 确保标签模板配置正确

### 技术支持

如果遇到问题，请检查：
1. Odoo日志文件
2. Python错误信息
3. 模块依赖关系

## 使用说明

安装完成后，您可以：
1. 在发货单中创建发货标签
2. 配置标签模板
3. 打印和管理标签
4. 跟踪标签状态

详细使用说明请参考模块内的帮助文档。
