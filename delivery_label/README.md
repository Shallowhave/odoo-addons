# 发货标签模块 (Delivery Label)

适用于 Odoo 18 的发货标签打印和管理系统。

## 功能特性

- 🏷️ **标签创建**：快速创建发货标签，支持多种模板
- 📋 **模板管理**：灵活的标签模板配置，适应不同需求
- 🖨️ **打印功能**：支持PDF打印，多种标签格式
- 📱 **二维码支持**：自动生成二维码，便于跟踪
- 📊 **状态管理**：完整的标签生命周期管理
- 🔗 **集成发货单**：与Odoo发货单无缝集成

## 安装要求

- Odoo 18.0 或更高版本
- Python PIL/Pillow 库
- Python qrcode 库
- 库存管理模块
- 销售模块（可选）

## 安装步骤

1. 将模块文件复制到 Odoo 的 addons 目录
2. 在 Odoo 中更新应用列表
3. 安装 "发货标签" 模块
4. 配置用户权限

## 使用方法

1. 在发货单中点击"创建发货标签"
2. 选择标签模板和数量
3. 系统自动生成标签编号和二维码
4. 打印标签并粘贴到包裹上
5. 更新标签状态跟踪发货进度

## 标签模板

- **标准标签**：100x60mm，包含完整信息
- **小标签**：80x50mm，适用于小包裹
- **批量标签**：48x30mm，一页多个标签
- **横向标签**：120x80mm，宽标签格式

## 权限配置

- **发货标签用户**：可以创建、查看和打印发货标签
- **发货标签管理员**：可以管理发货标签模板和所有发货标签
