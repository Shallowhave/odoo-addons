发货标签模块

适用于 Odoo 18 的发货标签打印和管理系统

🚀 主要功能

  • 标签创建：快速创建发货标签，支持多种模板
  • 模板管理：灵活的标签模板配置，适应不同需求
  • 打印功能：支持PDF打印，多种标签格式
  • 二维码支持：自动生成二维码，便于跟踪
  • 状态管理：完整的标签生命周期管理
  • 集成发货单：与Odoo发货单无缝集成

📋 标签模板

  • 标准标签：100x60mm，包含完整信息
  • 小标签：80x50mm，适用于小包裹
  • 批量标签：48x30mm，一页多个标签
  • 横向标签：120x80mm，宽标签格式

🎨 自定义选项

  • 显示/隐藏公司Logo
  • 显示/隐藏二维码和条形码
  • 显示/隐藏重量、体积信息
  • 显示/隐藏收货地址和产品列表
  • 字体大小调整
  • 标签方向设置（纵向/横向）

📊 状态管理

  • 草稿：新创建的标签
  • 已打印：标签已打印
  • 已发货：包裹已发出
  • 已送达：包裹已送达
  • 已取消：标签已取消

🔧 技术特性

  • 基于Odoo 18开发
  • 支持PIL图像处理
  • 集成QR码生成
  • 响应式设计
  • 多语言支持
  • 权限控制

📱 使用流程

  1. 在发货单中点击"创建发货标签"
  2. 选择标签模板和数量
  3. 系统自动生成标签编号和二维码
  4. 打印标签并粘贴到包裹上
  5. 更新标签状态跟踪发货进度

🛠️ 安装要求

  • Odoo 18.0 或更高版本
  • Python PIL/Pillow 库
  • Python qrcode 库
  • 库存管理模块
  • 销售模块（可选）
