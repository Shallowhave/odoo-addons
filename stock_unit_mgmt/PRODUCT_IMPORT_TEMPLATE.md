# 产品导入模板

本文档包含产品导入所需的字段说明，包括Odoo标准字段和自定义字段。

## 使用说明

1. 在Odoo中进入：**库存 > 产品 > 产品**
2. 点击**导入**按钮
3. 下载模板或使用本文档中的字段列表
4. 填写数据后上传

## Excel模板文件

已创建Excel模板文件：**`产品导入模板.xlsx`**

**文件位置：** `/opt/custom/addons/stock_unit_mgmt/产品导入模板.xlsx`

**模板包含：**
- **字段说明工作表**：包含所有字段的详细说明、数据类型、示例值
- **示例数据工作表**：包含3条示例数据（成品膜、配液原料、原膜）

**使用方法：**
1. 下载Excel模板文件
2. 在"示例数据"工作表中填写您的产品数据
3. 复制"示例数据"工作表中的数据（包括表头）
4. 在Odoo中：**库存 > 产品 > 产品 > 导入**
5. 上传Excel文件或粘贴数据
6. 映射字段并确认导入

## 字段说明

### 产品类型（type）
- `product`：可库存产品（可以入库、出库）
- `service`：服务类产品（不能入库）

### 产品类别（categ_id）
格式：`父类别 / 子类别`，例如：`All / Saleable`

### 计量单位（uom_id, uom_po_id）
使用单位名称，例如：`件(Units)`、`千克(kg)`、`升(L)`、`米(m)`

### 批次/序列号追踪（tracking）
- `none`：不追踪
- `lot`：批次追踪
- `serial`：序列号追踪

### 布尔值字段
- `True` 或 `1`：是/启用
- `False` 或 `0`：否/禁用

### 安全库存规则（safty_rule）
- `not_note1`：不包括备注1
- `not_note2`：不包括备注2
- `all`：包括所有
- `not_all`：都不包括

### 附加单位模板（default_unit_config）
- `kg`：公斤(kg)
- `roll`：卷
- `barrel`：桶
- `box`：箱
- `bag`：袋
- `sqm`：平方米(㎡)
- `custom`：自定义（需要填写`quick_unit_name`）

---

## 注意事项

1. **必填字段**：`name`、`type`、`categ_id`、`uom_id`、`uom_po_id` 为必填
2. **单位一致性**：注意各个字段的单位，确保数据正确
3. **计算字段**：`面积 (㎡)`和`体积 (cm³)`是自动计算的，不需要填写
4. **产品类型匹配**：成品膜填写 `finished_density` 和 `weight_per_sqm`，配液原料填写 `solution_solid_content` 和 `solution_viscosity`
5. **附加单位**：只有启用 `enable_custom_units` 后，附加单位配置才会生效
6. **数据格式**：小数使用小数点（.），布尔值使用 `True`/`False` 或 `1`/`0`

## 导入步骤

1. 准备Excel或CSV文件，使用上述字段名作为列标题
2. 填写产品数据
3. 在Odoo中：**库存 > 产品 > 产品 > 导入**
4. 上传文件
5. 映射字段（Odoo会自动识别大部分字段）
6. 确认导入

## 常见问题

### Q: 如何获取产品类别的完整路径？
A: 在Odoo中查看产品类别时，格式为 `父类别 / 子类别`，直接复制即可。

### Q: 如何获取计量单位的正确名称？
A: 在Odoo中：**库存 > 配置 > 计量单位**，查看单位名称，格式通常为 `单位名称(单位代码)`。

### Q: 导入后如何验证数据？
A: 导入后检查：
- 产品基本信息是否正确
- 自定义字段是否已填写
- 计算字段（面积、体积）是否正确计算
- 附加单位配置是否生效
